# smartmeter-back

Backend for Smart-metering on Nest.js

## Installation

```bash
$ npm install
```

## Start

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testing

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
